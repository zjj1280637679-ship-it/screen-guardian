# Scenario-Card Forest

Scenario cards turn the whitepaper into concrete user situations.

They are not tutorials and they are not full tests. Each card connects a real task to:

- user situation
- external conditions
- recommended route
- guard strategy
- context budget
- risks and fallback paths
- acceptance checks
- related whitepaper claims

## Forest Layout

| Branch | Purpose |
| --- | --- |
| `core/` | First-use and default perception tasks. |
| `advanced/` | Optional, routed, chained, or multi-step workflows. |
| `failure-modes/` | Known ways perception can mislead the AI. |
| `user-types/` | Distinct demand profiles such as beginner or high-intensity users. |
